# Tidal RPC
Simple Node.JS script for [Tidal](https://tidal.com/) RPC

## Installation

Use [git](https://git-scm.com/) to clone the package

```bash
git clone https://github.com/NotII/Tidal-RPC
cd Tidal-RPC
npm i
```

## Usage

```
node index.js
```
